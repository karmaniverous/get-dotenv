    Module dotenv

    Dotenv expansion helpers. Exported for consumers that need string and record expansion semantics compatible with get-dotenv (including defaults and recursive expansion).

    This module also exports format-preserving dotenv edit utilities under ./edit for in-place updates that preserve comments, whitespace, ordering, and unknown lines.
